### Introduction to Forms
Objectives:
- Use the < form ></ form > tag;
- Use the < input > tag;
- Use < label > </ label > tag;
- Write simple validations.

```html
<form action="/my-form-page" methoud="post">
<!-- All our inputs will go in here -->
</form>

Input examples..
<input type="text">
<input type="date">
<input type="color">
<input type="file">
<input type="checkbox">

```

- **Action**: the URL to send form data to;
- **method**: The type of HTTP request.
